# Time Complexity Analysis

| Algorithm | Complexity |
|------------|------------|
| Dijkstra | O((V+E)logV) |
| Floyd-Warshall | O(V³) |
| Kruskal MST | O(E log E) |
| Activity Selection | O(n log n) |
| Knapsack | O(nW) |
| Branch and Bound | Exponential |
| Priority Queue | O(log n) |
| Merge Sort | O(n log n) |
| Huffman Coding | O(n log n) |
